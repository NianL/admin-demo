:default-text        选填        string          对应data中的text，错误或者未填写默认选中第一个
    :default-value       选填        string          对应data中的value，同default-text，二者选择一个设置即可
    :data                必填        Array           格式 [{text:"",value:""}]
    @menu-click          必填        function        钩子，菜单选中以后返回信息
