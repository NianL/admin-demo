element：http://element-cn.eleme.io/#/zh-CN/component/
Basic
Layout 布局: 通过基础的 24 分栏，创建页面布局(代码简洁使用方便，推荐使用)
Container 布局容器: 不推荐使用，使用并不方便，自带有固定的样式，实际使用依然需要自己调整，如果会自己写css就自己写吧
Color 色彩: 主色(主题色)、辅助色(提示信息等)、中性色(文本和边框等)
Typography 字体: 没用过
Icon 图标: 可以单独使用，也能在button中配合使用，缺点就是图标太少了
Button 按钮: 一些常用的按钮，各种颜色，各种形状，统一美观，适合大部分场景使用
From
Radio 单选框、Checkbox 多选框: 和原生的功能差不多
Input 输入框: 用法很多，但是在表单验证的时候自带的一些方法会有一些莫名的bug
InputNumber 计数器: 数字固定增加，但是不能限制数值，比如只允许输入5的倍数，也不能禁止输入
Select 选择器: 功能很多
Cascader 级联选择器: 带有级联关系的选择框
Switch 开关: 开关切换
Slider 滑块: 设置百分比或者数值等使用的，左右拖动类似音量选择
TimePicker 时间选择器、DatePicker 日期选择器、DateTimePicker 日期时间选择器: 推荐
Upload 上传: 很好用，可以用自带的样式，也可以根据钩子返回信息自己定义上传样式和进度显示
Rate 评分: 选择小星星的评分组件
ColorPicker 颜色选择器: 颜色选择组件
Transfer 穿梭框: 左右移动数据
Form 表单: 样式不错，但是验证方面坑很多，建议用自己定义的方法(validator)
Data
Table 表格: 数据列表展示
Tag 标签: 用于标记和选择
Progress 进度条: -
Tree 树形控件: 树形控件
Pagination 分页: 分页组件
Badge 标记: 出现在按钮旁边的数字
Notice
Alert 警告: 做为页面显示使用的，放在某个dom显示的，不是弹出层
Loading 加载: 使用方便，在ie浏览器会存在一直显示的问题，分析原因是渲染问题，设置值为true以后不能马上设置为false(接口反应很快的时候)，解决方法加一点延迟或者nextTick
Message 消息提示: 页面顶部弹出消息，自动关闭的
MessageBox 弹框: 可以作用提示框，确认框
Notification 通知: 页面角落出现的提示框
Navigation
NavMenu 导航菜单: 网站的导航菜单
Tabs 标签页: 分隔页面多个页签
Breadcrumb 面包屑: 显示当前页面的路径，快速返回之前的任意页面
Dropdown 下拉菜单: 下拉展开的菜单
Steps 步骤条: 步骤条
Others
Dialog 对话框: 可以嵌套组件等复杂内容的弹出层
Tooltip 文字提示: 鼠标放上去显示的提示信息
Popover 弹出框: 和Tooltip类似，但是里面内容可以是组件等复杂的内容
Card 卡片: 将内容显示在一块区域，一个小卡片上，看起来就是一个小模块
Carousel 走马灯: 图片轮播
Collapse 折叠面板: 折叠显示内容，手风琴一样的
